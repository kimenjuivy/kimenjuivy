### Projects

#### 🏥 [Nairobi Healthcare Access Inequality Analysis](https://github.com/kimenjuivy/nairobi-healthcare-access-analysis)
> Built a Healthcare Access Inequality Index (HAII) for all 17 Nairobi constituencies. Combines facility density, KEPH-weighted facility levels, and population pressure — normalised 0–100. Includes interactive choropleth maps and formal policy recommendations.

| Finding | Value |
|:---|:---|
| Constituencies below 2 facilities per 10,000 | 11 of 17 |
| Level 4+ hospitals held in just 5 constituencies | 75.7% |
| Mathare facility density vs Nairobi average | 2.7× below |
| Nairobi population living in underserved areas | 71.2% |

`Python` `Pandas` `GeoPandas` `Folium` `Matplotlib` `Seaborn` `KMHFL 2017` `KNBS 2019`
