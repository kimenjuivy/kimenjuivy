## Projects

### [Nairobi Healthcare Access Inequality Analysis](https://github.com/kimenjuivy/nairobi-healthcare-access-analysis)

Built a Healthcare Access Inequality Index (HAII) from scratch using 2017 KMHFL facility data and 2019 KNBS population estimates. The index combines facility density, KEPH-weighted facility levels, and population pressure — normalised to a 0–100 scale. Includes spatial choropleth mapping, confidence-tiered data reconstruction for constituencies with missing census data, and formal policy recommendations addressed to Nairobi County Government.

**Key findings**

| Finding | Value |
|---|---|
| Constituencies below 2 facilities per 10,000 people | 11 of 17 |
| Level 4+ hospitals held in just 5 constituencies | 75.7% |
| Kasarani population with zero Level 4+ hospital access | 780,656 |
| Mathare facility density vs Nairobi average | 2.7× below |

**Stack:** Python · Pandas · GeoPandas · Folium · Matplotlib · Seaborn · KMHFL · KNBS 2019 Census · HDX GeoJSON
