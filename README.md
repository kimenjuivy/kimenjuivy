> Built a **Healthcare Access Inequality Index (HAII)** from scratch using 2017 KMHFL facility data and 2019 KNBS population estimates. The index combines facility density, KEPH-weighted facility levels, and population pressure — normalised to a 0–100 scale. Includes interactive spatial choropleth maps, confidence-tiered data reconstruction for constituencies with missing census data, and formal policy recommendations addressed to Nairobi County Government.
