## `$ cat ./projects/featured`

### 🏥 [Nairobi Healthcare Access Inequality Analysis](https://github.com/kimenjuivy/nairobi-healthcare-access-analysis)

> Built a Healthcare Access Inequality Index (HAII) from scratch using 2017 KMHFL facility data and 2019 KNBS population estimates. Combines facility density, KEPH-weighted facility levels, and population pressure into a normalised 0–100 scale. Includes spatial choropleth mapping, confidence-tiered data reconstruction, and formal policy recommendations addressed to Nairobi County Government.

```
HEADLINE FINDINGS
─────────────────────────────────────────────────────────
  11 of 17 constituencies  →  below 2 facilities per 10,000 people
  75.7% of Level 4+ hospitals  →  concentrated in just 5 constituencies
  Kasarani (pop. 780,656)  →  zero Level 4+ hospital access
  Mathare facility density  →  2.7× below the Nairobi average
─────────────────────────────────────────────────────────
```
